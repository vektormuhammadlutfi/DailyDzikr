# Daily Dzikr - Aplikasi Dzikir Pagi dan Petang

[![GitHub Stars](https://img.shields.io/github/stars/vektormuhammadlutfi/DailyDzikr?style=social)](https://github.com/vektormuhammadlutfi/DailyDzikr)
[![GitHub Forks](https://img.shields.io/github/forks/vektormuhammadlutfi/DailyDzikr?style=social)](https://github.com/vektormuhammadlutfi/DailyDzikr)

Aplikasi sederhana untuk membantu umat Muslim dalam menjalankan dzikir pagi dan petang sesuai sunnah.

## Deskripsi

Daily Dzikr adalah aplikasi web yang dirancang untuk memudahkan pengguna dalam melaksanakan dzikir pagi dan petang. Aplikasi ini menyediakan teks dzikir dalam bahasa Arab, transliterasi, dan terjemahan, serta dilengkapi dengan penjelasan tentang keutamaan (faedah) dari setiap dzikir.

## Fitur

-   Teks dzikir pagi dan petang yang lengkap
-   Tampilan dalam bahasa Arab, transliterasi, dan terjemahan
-   Penjelasan tentang keutamaan setiap dzikir
-   Pengaturan tampilan (mode gelap/terang, ukuran font, dll.)
-   Progress bar untuk memantau penyelesaian dzikir
-   Progressive Web App (PWA) - dapat diinstal di perangkat Android dan iOS

## Tech Stack

-   **React**: Library JavaScript untuk membangun antarmuka pengguna
-   **Vite**: Build tool yang cepat untuk pengembangan web modern
-   **Tailwind CSS**: Framework CSS untuk styling yang cepat dan responsif
-   **Lucide React**: Library ikon
-   **Date-fns**: Library untuk memformat tanggal
-   **Sonner**: Library untuk notifikasi toast
-   **PWA**: Progressive Web App

## Cara Penggunaan

1.  Clone repositori ini:

    ```bash
    git clone https://github.com/vektormuhammadlutfi/DailyDzikr.git
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Jalankan aplikasi:

    ```bash
    npm run dev
    ```

    Buka aplikasi di browser Anda.
